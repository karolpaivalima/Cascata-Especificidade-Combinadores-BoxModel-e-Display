Regra da Cascata

A regra da cascata no CSS afirma que, uma vez que o arquivo é lido como se fosse uma cascata, em caso de conflito entre as propriedades, a última ordenança é a que deve prevalecer.

Especificidade

Diferentementemente da regra da cascata, a especificidade (que nada mais é do que o quão específico ou qual a quantidade de pontos que cada seletor tem) conduz o código a considerar como mais importante, e portando executar o comando, mais específico. A diferença entre o seletor pela tag, pela classe e pelo id. O id é o seletor mais especifico, a classe é menos específica que o id porém mais específica do que a tag, e a tag é a mais geral (e menos específica de todas entre essas).

Diferenças entre display inline e display block

Um display block sempre ocupa a largura da tela inteira então o próximo elemento colocado se localizará logo embaixo. Diferentemente do display inline, no qual os elementos se agrupam lado a lado, pois eles só ocupam o espaço necessário para existir e em elementos com display inline, width e height não se aplicam.

Propriedades que podem ser aplicadas no display: block

  • width;
  • height;
  • padding;
  • margin;
  • border.

Propriedades que podem ser aplicadas no display: inline

  • padding;
  • margin;
  • border.

Elementos que são por padrão display inline:

  • a;
  • strong;
  • span;
  • img.

Elementos que são por padrão display block:

  • h1;
  • p;
  • div;
  • section;
  • main;
  • header;
  • nav.
